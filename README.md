# Apollo & Redux Example
```
 - NextJs
 - Redux
 - Apollo
 - GraphQl
 - Rebass
 - React-motion
```
## How to use

Install it and run:

```bash
npm install
npm run dev
```

Deploy it to the cloud with [now](https://zeit.co/now) ([download](https://zeit.co/download)):

```bash
now
```
